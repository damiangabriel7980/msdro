(inner) sendNotification(template_name, template_content, to, subject, changesNotification, confDate, confHour, confName, userStatus, spkString, conferencesLink, mergeVars)

Send email via mandrill API (used for MSD Conferences which is no longer active)
Parameters:
Name Type Description
template_name String The name of the Mandrill template
template_content Array an array of objects which contain custom fields to be filled (ex. : {name: 'varName', content: 'someContent'})
to String The recipient email address
subject String The subject of the email
changesNotification Boolean If true, a message regarding changes of a conference will be sent. Otherwise, an invitation to a conference email will be sent
confDate String The date of the conference
confHour String The hour of the conference
confName String The name of the conference
userStatus String The status of the user that receives the email
spkString String A string containing a list of speakers that attend the conference
conferencesLink String A link to the conference
mergeVars Array Other Mandrill API custom variables to send (ex. [ { "rcpt": "emailadress@domain.com", "vars": [ { "name": "fname", "content": "John" }, { "name": "lname", "content": "Smith" } ] } ])
